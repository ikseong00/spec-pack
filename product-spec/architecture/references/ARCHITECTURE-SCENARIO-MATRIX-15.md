# Architecture Scenario Matrix 15

이 문서는 architecture pack loop에서 우선 검증할 scenario set이다.

1. 규제성 클리닉 intake + 예약 + 재방문
2. 지역 법률 상담 intake + delegate + callback
3. 보험 서류 업로드 + 심사 대기 + 이의제기
4. 이커머스 반품/교환 approval queue
5. 정부 민원 접수 + 처리 상태 + 담당자 이관
6. 현장 점검 모바일 앱 + 오프라인 체크리스트 + 증빙 사진
7. SaaS self-serve to enterprise admin controls + security gate
8. API-first 물류 sync dashboard + incident history
9. 생산 라인 이상 신고 + review + escalation dashboard
10. 창고 출고 작업자 앱 + scanner + sync retry
11. 구인구직 매칭 + review + moderation
12. 크리에이터 membership + progression + paywall
13. 투자 리서치 membership + citation freshness
14. 공장 설비 유지보수 dispatch + parts shortage recovery
15. 동네 빵집 발견 지도 + 저장 + 제보

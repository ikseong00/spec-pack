# Design Scenario Matrix 50

이 문서는 design Ralph loop에 사용할 50개 시나리오 배치다.

각 시나리오는 산업/분야와 product shape를 같이 섞는다.

   - stress: pickup state, calendar, order confirmation
3. 식품 제조사 B2B 재고/발주 포털
   - shape: b2b workflow
   - stress: dense data screens, role split, table clarity
4. 카페 운영자용 원두 정기구독 SaaS
   - shape: self-serve SaaS

5. 지역 맛집 큐레이션 멤버십 서비스
   - shape: content membership

## Use Rule

- 한 loop에서 30-50개를 batch로 묶는다
- 매 loop마다 runner는 domain-specific answer를 실제 founder/operator처럼 제공한다
- evaluator는 산업이 아니라 output quality와 design criteria를 본다
- 특정 산업에만 맞는 fix가 나오면 reusable contract로 다시 일반화한다

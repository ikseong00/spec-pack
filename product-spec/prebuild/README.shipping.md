## Boundary

- planning fact를 design이 다시 invent하지 않는다
- shared doc 충돌은 owning doc로 되돌린다
- rough architecture는 큰 구조와 제약만 잠근다
- API/component/module 설계는 개발자 책임이다

## Excluded By Default

- internal authored bundles
- worked examples
- loop log
- loop protocol / scenario matrix / loop memory

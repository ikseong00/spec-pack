# Prebuild Worked Examples

이 디렉터리는 `planning + design` 통합 pack을 어떻게 쓰는지 보여주는 공개 가능한 example set이다.

원칙:

- internal fixture 전체를 그대로 내보내지 않는다
- route, required fields, cross-doc projection만 보여준다
- source of truth는 아니다
- hard-case에서 어떤 owner/value가 중요했는지 빠르게 이해시키는 용도다

- [legal-intake-delegate-callback.md](legal-intake-delegate-callback.md)

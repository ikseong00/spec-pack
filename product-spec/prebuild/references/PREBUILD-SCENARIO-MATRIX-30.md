# Prebuild Scenario Matrix 30

이 문서는 planning + design 통합 Ralph loop에 사용할 30개 시나리오 배치다.

각 시나리오는 planning ambiguity와 design translation risk가 함께 보이도록 골랐다.

1. 동네 빵집 발견 지도 + 저장 + 제보
2. 규제성 클리닉 intake + 예약 + 재방문
3. 건축 fit-out 승인 + 현장 증빙 + 변경요청
4. 크리에이터 membership + progression + paywall
5. 배달권역 기반 세탁 수거 예약 + 상태 추적
6. 생산 라인 이상 신고 + review + escalation dashboard
7. 학원 숙제/채점/재채점 포털
8. 지방 소상공인 지원금 신청 포털
9. 지역 법률 상담 intake + delegate + callback
10. 보험 서류 업로드 + 심사 대기 + 이의제기
11. 중소 병원 staff scheduling + no-show recovery
12. 현장 점검 모바일 앱 + 오프라인 체크리스트 + 증빙 사진
13. B2B vendor onboarding + 문서 검수 + 승인 상태
14. API-first 물류 sync dashboard + incident history
15. 공장 설비 유지보수 dispatch + parts shortage recovery
16. 투자 리서치 membership + citation freshness
17. 이커머스 반품/교환 approval queue
18. 로컬 부동산 listing + claim + freshness explanation
19. 여행 itinerary AI planner + booking handoff
20. SMB CRM + pipeline + delegated assistant view
21. 학교 공결/증빙 제출 + 승인/반려
22. 구인구직 매칭 + review + moderation
23. 이벤트 티켓 waitlist + release + refund
24. 카페 프랜차이즈 운영 대시보드 + 알림 밀도
25. 정부 민원 접수 + 처리 상태 + 담당자 이관
26. 창고 출고 작업자 앱 + scanner + sync retry
27. 뉴스레터 아카이브 + paid member tier + support
28. 회계 자료 수집 + reviewer approval + audit trail
29. 대학 프로젝트 피드백 + rubric + regrade
30. SaaS self-serve to enterprise admin controls + security gate
